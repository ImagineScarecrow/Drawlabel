创建虚拟环境
conda create -n label python=3.6.2
安装环境
pip3 install -r requirements.txt




可执行程序 
可执行文件放在 exe_file文件夹
1、在linux环境下运行 DrawLabel_v0.1 
2、在windows环境下运行DrawLabel_v0.1 .exe

v0.1修改
1、打开文件后按下一页闪退的问题
2、删除掉标签列别 列表框
3、修正已标注数量，在再一次打开文件夹后出现累加的情况。
4、添加标签框保存提示语
5、修改了图片自适应窗口大小功能。

v0.2
1、新增矩形框标注（矩形框类别就用位置ID来代替）
2、新增拖拽功能（放大之后才可以拖拽）
3、新增图片适应窗口大小功能
4、新增适应窗口按键（图片放大后自适应会失效，可以通过适应窗口按键来恢复）
5、修改了进度显示框，显示已标注文件数/总数 通过按键可跳转到最近一次标注的文件处
6、标注到一半未保存时想要退出可以按esc键
7、标注点标错可以退出标记后手动调节标注点位置。

v0.2.1
1、将删除，折线，矩形框，退出标注变成下拉列表形式。
2、删除添加delete快捷键。
3、修改进度显示控件，始终显示编号最高的哪个已完成标注文件
4、将删除按键设置为只有选中标签的前提下才能触发

v0.3
1、修改了标注模式转换，无需事先退出标注，直接转换
2、修改了进度显示，显示当前文件在整个列表中的位置，通过输入数字跳转到对应文件，
输入图片名跳转到对应图片。
3、增加配置文件，将所有标签以文字描述，保存到json中会转为对应的数字。
4、增加撤销最近标注点的功能，按Ctrl+z 可撤销

v0.4
1、增加culane格式标签的存储，显示，和编辑
2、增加右键退出标注功能。

v0.5
1、增加点击右边标签删除标签功能。

v.06
1、新增文件删除撤回功能，删除图片后可按ctrl+z 撤回
2、目前默认显示.lines.txt文件，在配置文件exe_file/config.json (如果运行代码 就修改主文件夹的config.json文件)中可以修改 __TXT_SUFFIX__ 来决定显示.txt 文件或者.lines.txt文件
3、新增点删除功能，鼠标对准要删除的点，按x可删除对应点，目前暂不支持撤回功能。
4、修改了连续删除标签闪退bug
